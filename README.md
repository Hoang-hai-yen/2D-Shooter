## 📖 Description

**2D Shooter** is a top-down action game where the player fights through levels filled with AI-driven enemies, hidden chests, and locked doors. The game blends fast-paced combat with light puzzle elements — find the key hidden among multiple chests to unlock the exit and advance to the next level.

The final level culminates in a dedicated **Boss Battle** with unique mechanics and significantly higher HP.

---

## 🎮 Gameplay

- **Move & Aim**: 8-directional movement (WASD) + free 360° mouse aiming
- **Combat**: Shoot down patrolling, chasing, and cover-seeking AI enemies
- **Puzzle**: Find the one chest containing the hidden **Key** among many decoy chests
- **Progress**: Use the Key to open the exit door and advance to the next level
- **Boss Fight**: Level 3 is a dedicated boss arena — survive and win

---

## 🕹️ Controls

| Action | Input |
|--------|-------|
| Move | `WASD` |
| Aim & Shoot | Mouse |
| Interact (open chest/door) | `E` |
| Pause / Back to Menu | `Q` |

---

## ✨ Core Features

- **AI Enemy System** — enemies patrol, chase, and take cover dynamically
- **True/Fake Chest System** — only one chest holds the key; exploration is required
- **Weapon Drop System** — defeated enemies drop guns the player can pick up
- **Dual-Mode Guns** — Burst mode and Shotgun mode with different fire patterns
- **Boss Battle Level** — Level 3 features a dedicated boss with unique behavior and high HP
- **Health System** — collect health potions dropped in the level to recover HP
- **Auto-pickup** — weapons and health items are picked up automatically on contact
- **Win / Lose States** — "You Lose" panel with Retry and Main Menu options
